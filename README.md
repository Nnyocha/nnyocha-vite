# Nnyocha Platform

**Nnyocha** is a research-driven digital platform designed to connect **researchers, investors, and mentors** within a unified innovation ecosystem.

The platform provides an environment where ideas can be shared, validated, funded, and scaled — bridging the gap between research and real-world impact.

🌐 Live Platform: https://nnyocha.com

---

## 🚀 Overview

Nnyocha is built to:

* Enable researchers to publish and showcase work
* Provide visibility to innovative ideas
* Connect investors with high-potential research
* Facilitate mentorship and collaboration
* Support knowledge sharing through a blog system

---

## ✨ Current Features

* 🏠 Landing page (core platform introduction)
* 📝 Blog system (content publishing & reading)
* 📡 API-driven backend structure
* ⚙️ Scalable architecture for future modules

## 🛠️ Tech Stack

### Frontend

* React (Vite)
* Tailwind CSS
* Framer Motion (animations)
* React Router
* Swiper (UI components)
* Markdown rendering (react-markdown, marked)
* Email integration (EmailJS)

### Backend

* Node.js (ES Modules)
* Express.js
* MongoDB (Mongoose)
* Cloudinary (media storage)
* Multer (file uploads)
* Nodemailer (email services)
* Rate limiting & security middleware

## ⚙️ Installation & Setup

### 1. Clone the repository

```
git clone https://github.com/Nnyocha/nnyocha-vite.git
cd nnyocha-vite
```

---

### 2. Frontend Setup

```
cd frontend
npm install
npm run dev
```

---

### 3. Backend Setup

```
cd backend
npm install
npm run dev
```

---

## 🔐 Environment Variables

Create a `.env` file in the backend directory:

```
PORT=5000
MONGO_URI=your_mongodb_connection
CLOUDINARY_URL=your_cloudinary_url
EMAIL_USER=your_email
EMAIL_PASS=your_password
```
